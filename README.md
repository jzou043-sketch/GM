# Genomic Mating (GM)

This repository contains code, processed data, simulation outputs, real-data analyses, robustness analyses, and documentation for the genomic mating project.

## Main contents

- GM.R: main genomic mating R code.
- GM_pipeline_user_guide.pdf: user guide for the GM pipeline.
- data/: GM-ready datasets.
- Performance of different breeding strategies under varying dominance scenarios/: dominance-scenario simulation outputs.
- Comparative performance of optimization algorithms in a simulation/: optimizer comparison outputs.
- Multi-generation simulation design and overall results/: multi-generation simulation outputs.
- Chicken real-data analysis/: chicken real-data analysis outputs.
- Mouse real-data analysis/: mouse real-data analysis outputs.
- Pig real-data analysis/: pig real-data analysis outputs.
- Pair-level progeny advantage analysis/: pair-level progeny advantage outputs.
- Robustness of GM optimization few starting points and moderate candidate pools are sufficient/: robustness analyses for n_restart and candidate-pool size.

Large data and high-resolution figures are tracked with Git LFS.
