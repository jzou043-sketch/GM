Matrix-first complementary-QTL simulation with moderate GM advantage
===============================================================

This dataset was generated to compare PS, GS, GS+GM, and GM under four dominance scenarios.
The first 2,000 SNPs are effective QTL and the remaining 8,000 SNPs are background markers.
All four scenarios use the same SNP genotypes but different dominance-effect strengths.
Each scenario is scaled so that PS has a total objective value of 1000.

GM-ready files:
S0_no_dominance_gm_dat.csv
S1_weak_gm_dat.csv
S2_moderate_gm_dat.csv
S3_strong_gm_dat.csv
